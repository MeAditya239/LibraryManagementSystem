📚 Library Management System
A Java-based web application to manage a library's book inventory and member interactions.


🚀 Features

👨‍💼 Admin Panel
        Issue and return books
        Manage book inventory (add, update, delete)

🙋‍♂️ Member Login
        Login with credentials
        Search and view available books


🛠️ Technologies Used
        Java
        JSP & Servlets
        MySQL
        HTML/CSS (for basic UI)
